Implementation of the Repository pattern for storing, retrieving and deleting Aggregates.

A Repository instruments an EventStore to:

  • Insert Events in the EventStore for an Aggregate, using the AggregateRoot,
  • Get all the Events in the EventStore and rebuild the State of an Aggregate, into a new AggregateRoot instance,
  • Remove all the Events for an Aggregate in the EventStore.
