Command Handler trait referring to Aggregate with Option state, a.k.a. Aggregate.

Implementations of this trait can be adapted back into the command::Handler foundation trait by using as_handler, in cases where the implementation has compile-time known size.

Associated Types

type Command

Commands to trigger a specific use-case on the context of an Aggregate.

Most often than not, this type should be an enum containing all supported operations -- that are not queries -- for the specified Aggregate.

type Aggregate: Aggregate

Domain entity produced, updated or, in some way, affected by a Command.

type Error

Possible expected errors to be returned when handling a Command fails.
